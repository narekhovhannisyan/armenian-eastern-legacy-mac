## armenian-eastern-legacy-mac

Armenian Eastern (Legacy) keyboard for MacOS created with Ukelele.

# Installing

1. Download and install [Ukelele](https://software.sil.org/ukelele/).

2. Then download keyboard file by pressing `Clone or Download` button, then `Download ZIP`.

3. Double click `Armenian Eastern.bundle` to install keyboard layout.

4. Then go to `System Preferences` -> `Keyboard` -> `Input Sources`

5. Add `Armenian Eastern` from `Armenian` section.
